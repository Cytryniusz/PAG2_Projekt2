PAG - Projekt 2
Piotr Pawlus, Bartosz Ziółkowski, Szymon Ziędalski
Aplikacja do przetwarzania, wczytywania, analizowania i modelowania danych meteorologicznych, administracyjnych oraz astronomincznych.

Skrypt uruchamiany jest z poziomu terminala z podaniem odpowiednich argumentów.
Przykład uruchomienia:
python app1.py --year 2024 --month 3
